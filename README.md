### the idea

in the age of agentic ai, any thin wrapper around an api can be replicated in one session. openclaw was a wrapper. anthropic changed the terms. we changed the `$PATH`. the skills, state, configs, secrets вҖ” all stayed. only the binary that parses the prompt changed.

> *"never change what the code does вҖ” only how it does it."*
> вҖ” [code-simplifier](https://github.com/anthropics/claude-plugins-official/blob/main/plugins/code-simplifier/agents/code-simplifier.md)

**simplicity is portability. portability is sovereignty.**
